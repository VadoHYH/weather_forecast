## 📌 專案說明

本專案為天氣預報網站

1.提供當天每小時以及未來一周的天氣預報

2.提供當天的累積雨量(10分鐘、1小時、3小時、6小時、12小時、24小時)

3.依據氣溫與降雨機率，提供使用者衣著建議與天氣資訊。
